Introduction to Software Engineering by Samson Kapeyi

Posted on 2nd March,2024 by Akullo Ritah

Hello!

In this blog,I aim to demystify the intricacies of software engineering, exploring everything from fundamental concepts to advanced techniques, industry trends, and best practices which we learnt in the first week.These were; How the internet works,
How to build software,
What HTML means and its application,
What CSS means and its application,

My experiences after learning HTML & CSS were;

Building Static Websites: With HTML and CSS skills, I can start building my own websites from scratch. These could be personal blogs, portfolios, or small business websites.

Understanding Web Structure: I gained a deeper understanding of how web pages are structured and styled. This knowledge allows you to navigate and manipulate existing websites more effectively.

Responsive Design: Learning CSS enables you to create responsive designs that adapt to different screen sizes and devices. This is essential for ensuring a consistent user experience across desktops, tablets, and mobile phones.

Enhancing Existing Websites: I can make modifications and enhancements to existing websites, whether it's tweaking the layout, adding new features, or improving the overall design.

In the second week of studies,we were introduced to Git in the cloud where we learnt how; Git functions, its branches; to pull requests and merging etc.

We were also introduced to Bootstrap where we learnt how fast and flexible it eases coding.
